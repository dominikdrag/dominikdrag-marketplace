# dominikdrag-marketplace

Personal plugin marketplace for Claude Code.

## Installation

Add this marketplace (run from Claude Code CLI):

```
/plugin marketplace add dominikdrag/dominikdrag-marketplace
```

## Available Plugins

| Plugin | Description | Version |
|--------|-------------|---------|
| [bug-fix](https://github.com/dominikdrag/bug-fix) | Comprehensive bug investigation and fixing workflow with specialized agents for codebase exploration, root cause analysis, hypothesis testing, and fix validation | 1.0.2 |
| [feature-dev](https://github.com/dominikdrag/feature-dev) | Guided feature development with codebase understanding and architecture focus | 1.1.2 |
| [code-explain](https://github.com/dominikdrag/code-explain) | Deep, multi-layered code explanations with context gathering, history analysis, and pattern recognition | 1.0.5 |

## Installing Plugins

Run from Claude Code CLI:

```
/plugin install bug-fix@dominikdrag-marketplace
```
